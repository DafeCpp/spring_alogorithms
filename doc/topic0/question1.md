# Определение алгоритма. Примеры простых алгоритмов

Этот файл покрывает первый подпункт введения:

- определение алгоритма;
- вычисление числа Фибоначчи;
- проверка числа на простоту;
- быстрое возведение в степень.

## 1. Что такое алгоритм

Алгоритм — это конечная, точная и понятная последовательность действий,
которая по входным данным строит результат.

У алгоритма обычно есть 5 важных свойств:

1. `Дискретность` — решение разбито на шаги.
2. `Определённость` — на каждом шаге понятно, что делать.
3. `Конечность` — алгоритм должен завершаться.
4. `Результативность` — после завершения получается ответ.
5. `Массовость` — алгоритм применим не к одному примеру, а к целому классу задач.

## 2. Числа Фибоначчи

Определение:

- `F(0) = 0`
- `F(1) = 1`
- `F(n) = F(n - 1) + F(n - 2)`

Наивная рекурсия:

```cpp
int Fib(int n) {
  if (n <= 1) {
    return n;
  }
  return Fib(n - 1) + Fib(n - 2);
}
```

Почему это плохой алгоритм:

- много одинаковых подзадач;
- время `O(2^n)`;
- при росте `n` работает очень медленно.

Итеративная версия:

```cpp
int Fib(int n) {
  if (n <= 1) {
    return n;
  }

  int prev = 0;
  int cur = 1;
  for (int i = 2; i <= n; ++i) {
    int next = prev + cur;
    prev = cur;
    cur = next;
  }
  return cur;
}
```

Плюсы:

- время `O(n)`;
- память `O(1)`.

## 3. Проверка числа на простоту

Число простое, если у него ровно два делителя: `1` и само число.

Простейшая идея:

```cpp
bool IsPrime(int n) {
  if (n < 2) {
    return false;
  }
  for (int d = 2; d < n; ++d) {
    if (n % d == 0) {
      return false;
    }
  }
  return true;
}
```

Улучшение:

- достаточно проверять делители до `sqrt(n)`;
- если `d > sqrt(n)` и делителя не найдено, дальше искать уже бессмысленно.

```cpp
bool IsPrime(int n) {
  if (n < 2) {
    return false;
  }
  for (int d = 2; d * d <= n; ++d) {
    if (n % d == 0) {
      return false;
    }
  }
  return true;
}
```

Сложность:

- было `O(n)`;
- стало `O(sqrt(n))`.

## 4. Быстрое возведение в степень

Нужно вычислить `a^n`.

Наивный способ:

```cpp
long long Pow(long long a, int n) {
  long long result = 1;
  for (int i = 0; i < n; ++i) {
    result *= a;
  }
  return result;
}
```

Это `O(n)`.

Идея ускорения:

- если `n` чётное, то `a^n = (a^(n / 2))^2`;
- если `n` нечётное, то `a^n = a * a^(n - 1)`.

```cpp
long long FastPow(long long a, int n) {
  if (n == 0) {
    return 1;
  }
  if (n % 2 == 0) {
    long long half = FastPow(a, n / 2);
    return half * half;
  }
  return a * FastPow(a, n - 1);
}
```

Сложность:

- `O(log n)`.
